# 权限矩阵

## 角色定义

| 角色 | 说明 |
|---|---|
| 管理者 | 系统最高权限，拥有全部功能 |
| 教务老师 | 管理学生、售前、排课、购琴、物料、考核、活动和跟进任务 |
| 普通老师 | 只管理自己的课程、自己的学生和自己的复盘 |

## 关键原则

- 普通老师只看与自己有关的数据。
- 普通老师可以填写教学信息，但不建议看到完整商业成交判断。
- 教务老师负责转化、续课、购琴、物料和跟进。
- 管理者负责经营总览和系统配置。
